# Akuzipik-ASR
Code corresponding to the paper "Addressing Domain Mismatch in ASR for Akuzipik Language Documentation" in ComputEL-9, 2026

The code was adapted from https://huggingface.co/blog/mms_adapters to finetune an existing adapter for Meta's Massively Multilingual Speech ASR model.

Models and data are not made available in the interest of Indigenous data sovereignty for the Yupik people of St. Lawrence Island, AK.
